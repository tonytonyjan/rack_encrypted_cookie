# Cookie Structure

## `Rack::Session::EncryptedCookie`

```
+--------------------- uri encode --------------------+
| +------------- base64 ------------+      +-- hex -+ |
| | +-- base64 -+      +- base64 -+ |      |        | |
| | |  marshal  | "--" |    iv    | | "--" |  hmac  | |
| | +-----------+      +----------+ |      |        | |
| +---------------------------------+      +--------+ |
+-----------------------------------------------------+
```

## `Rack::Session::Cookie`

```
+--------- uri encode ----------+
| +-- base64 -+      +-- hex -+ |
| |  marshal  | "--" |  hmac  | |
| +-----------+      +--------+ |
+-------------------------------+
```
